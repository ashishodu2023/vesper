# VESPER for VS Code / Cursor

Thin scaffold that shells out to the `vesper` binary from your workspace folder.

## Install

1. Install the CLI: `cargo install --path ../../vesper-cli`
2. In VS Code / Cursor: **Extensions → … → Install from VSIX…**  
   or open this folder and press F5 (Extension Development Host)

## Commands

- **VESPER: Summarize Workspace**
- **VESPER: Fix (auto)**
- **VESPER: Run Task…**
- **VESPER: Doctor**

Output opens in the **VESPER** channel.
